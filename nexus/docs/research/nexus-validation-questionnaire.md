# Nexus Validation Questionnaire

## Design Principles Applied
- **Mom Test**: No leading questions, focus on PAST behavior not hypotheticals
- **JTBD**: Understand the job they're trying to accomplish
- **Pain Scoring**: Measure frequency + intensity + current spend
- **Clone + Feature**: Identify gaps in existing tools

---

## ANALYSIS FRAMEWORK

After collecting responses, analyze with Claude:

```
PROMPT FOR CLAUDE:

I collected [X] survey responses about automation pain points.
Analyze and give me:

1. TOP 5 PAIN POINTS (ranked by frequency + intensity)
2. COMMON COMPLAINTS about existing tools (Zapier, Make)
3. FEATURE GAPS (what people wish existed)
4. WILLINGNESS TO PAY distribution
5. EXACT QUOTES I can use in marketing
6. RECOMMENDED NEXUS POSITIONING based on this data
```

| Metric | Target | Why |
