## Tools by Use Case

### "I want to track competitor website changes"
1. Crayon - Best, expensive
2. Kompyte - Good, mid-price
3. Competitors.app - Basic, affordable

### "I want to analyze app store reviews"
1. AppFollow - Best features
2. Appbot - Simpler, good
3. Data.ai - Best data, very expensive

### "I want to mine B2B software reviews"
1. G2 Seller - Has the data, expensive
2. ReviewTrackers - Aggregates multiple
3. Birdeye - Good, but not CI-focused

### "I want to monitor social mentions"
1. Brandwatch - Best, very expensive
2. Brand24 - Good, affordable
3. Mention - Basic, cheap

### "I want to find product opportunities"
**NONE - THIS IS THE GAP**

## Pricing Tiers Analysis

### Enterprise ($1,000+/month)
- Klue, Crayon, Brandwatch, Talkwalker, Data.ai, Meltwater
- Features: Comprehensive, team collaboration, integrations
- Buyer: VP/Director level, funded companies

### Mid-Market ($200-999/month)
- Kompyte, Contify, Sprout Social, Birdeye, Similarweb, SEMrush, Ahrefs
- Features: Good functionality, some limitations
- Buyer: Marketing managers, growth teams

### SMB ($50-199/month)
- Appbot, AppFollow, ReviewTrackers, Brand24, Mention
- Features: Core functionality, some limits
- Buyer: Startup founders, small teams

### Starter ($0-49/month)
- Competitors.app, Owler, Mention Solo
- Features: Basic monitoring, limited
- Buyer: Indie hackers, bootstrappers

### Gap: Affordable + Powerful
- $29-79/month
- Review mining + AI insights
- Solo founder workflow
- NO tool exists here
