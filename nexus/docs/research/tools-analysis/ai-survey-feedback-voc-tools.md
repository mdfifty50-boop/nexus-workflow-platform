## CATEGORY 1: AI SURVEY & FORM BUILDERS

### 1. Typeform (AI Features)

**URL:** https://www.typeform.com

**What it does:**
- Conversational form builder with AI-powered question generation
- AI Formless: Natural language survey creation ("Build a customer satisfaction survey")
- AI-powered response analysis and insights
- Smart logic for conditional question branching
- Video and audio response collection

**AI Capabilities:**
- Generate entire surveys from prompts
- Analyze open-ended responses
- Smart follow-up question suggestions
- Sentiment detection in responses

**Pricing:**
- Basic: $25/month (limited AI features)
- Plus: $50/month
- Business: $83/month (full AI features)
- Enterprise: Custom

**API:** Yes - robust REST API, webhooks, Zapier/Make integrations

**Strengths:**
- Best-in-class UX for respondents
- Strong brand recognition
- Excellent completion rates (conversational style)
- Rich media support

**Weaknesses/Gaps:**
- AI features still emerging (not core product)
- Expensive for volume
- Limited deep analysis (surface-level AI)
- No interview transcription

**Integration Possibilities:**
- Zapier, Make, native integrations
- Webhook-based real-time data flow
- Embed in any website

## COMPETITIVE LANDSCAPE SUMMARY

### Market Positioning Matrix

| Tool | Survey Creation | AI Analysis | Price Point | Best For |
|------|----------------|-------------|-------------|----------|
| Typeform | Best UX | Basic | Medium | Brand-conscious |
| Qualtrics | Research-grade | Best | Enterprise | Large research teams |
| Dovetail | None | Excellent | Medium | Research repository |
| Viable | None | Good (GPT) | Medium | Quick insights |
| Sprig | In-product | Good | Medium | Product teams |
| SurveyMonkey | Standard | Good | Medium | General surveys |

### Gap Analysis for Ideation

**Unmet Needs (Opportunities):**

1. **AI Question Generation + Deep Analysis** - No tool does both exceptionally
2. **Affordable AI Analysis** - Most analysis tools are $500+/month
3. **Business Idea Validation Focus** - Tools are generic, not entrepreneurship-focused
4. **Regional Intelligence** - No tool has Kuwait/Gulf-specific intelligence
5. **Conversational Analysis to Questions** - No tool turns customer conversations into survey questions
6. **Idea-to-Validation Pipeline** - No tool guides from idea through validation
7. **Small Business Pricing** - Gap between free and $500+/month

### Recommended Integrations for Ideation

**Data Collection (Partner or Build):**
- Typeform API for beautiful forms
- Tally API for cost-effective forms
- Formbricks (open-source base)

**Analysis (Integrate):**
- OpenAI GPT-4 API (build custom)
- Deepgram for transcription
- Custom theme extraction

**Distribution (Integrate):**
- User Interviews for recruitment
- Respondent.io for B2B
- Social media APIs
