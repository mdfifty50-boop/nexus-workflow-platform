# Status Command

Generate Director status report for CEO.

## Instructions

1. Read `.claude-session.md` for current state
2. Read `_bmad-output/nexus-sprint/scope-document.md` for approved scope
3. Check todo list for progress

4. Report in format:

```

║                    DIRECTOR STATUS UPDATE                     ║
╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  CURRENT PHASE: [from scope document]                        ║
║                                                              ║
║  COMPLETED:                                                  ║
║  ✓ [Task] - [Result]                                        ║
║                                                              ║
║  IN PROGRESS:                                                ║
║  ◐ [Task] - [X]%                                            ║
║                                                              ║
║  PENDING:                                                    ║
║  ○ [Task]                                                    ║
║                                                              ║
║  CONTEXT USAGE: [X]%                                         ║
║  SCOPE ALIGNMENT: ✓ All work within approved scope          ║
║                                                              ║
║  CEO INTERVENTION: [REQUIRED / NOT REQUIRED]                 ║
╚══════════════════════════════════════════════════════════════╝
```
