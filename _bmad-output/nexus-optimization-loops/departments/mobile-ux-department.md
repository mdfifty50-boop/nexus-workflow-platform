## Department Mission

**Goal:** A user driving their car or walking in a mall can:
1. Open Nexus on mobile (< 2 seconds)
2. Tap microphone OR type quickly
3. Say/type request in English OR Kuwaiti Arabic
4. See workflow created instantly
5. Tap "Execute" once
6. Go about their day while Nexus works
7. Get notification when done

## Collaboration Protocol

When collaborating with main team:
- Mobile UX Dept works in PARALLEL, never blocking main team
- Integration points: UX reviews, voice testing, mobile validation
- Use Playwright MCP for mobile viewport testing
- Async handoffs via shared documents
